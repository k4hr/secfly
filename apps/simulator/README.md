# Виртуальная модель

Детерминированная упрощённая модель полностью синтетической среды. Поддерживает локальный маршрут, фиксированный шаг, паузу, сброс, виртуальный заряд и версионированные события. Это не физическая модель настоящего аппарата. Реальные координаты, устройства и эксплуатационные параметры запрещены.
