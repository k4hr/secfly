# Процессы ЭТАПА 1

- `ground-control`: русскоязычная оболочка Next.js без подключения к аппарату.
- `onboard-core`: Fastify-оболочка и `/health`; смена режимов отсутствует.
- `simulator`: Fastify-оболочка синтетической среды; физика отсутствует.
- `telemetry-gateway`: Fastify-оболочка для будущих синтетических сообщений.
- `scenario-runner`: CLI для проверки доступности каркаса; настоящие сценарии отсутствуют.

Процессы обмениваются только будущими проверенными сообщениями. На ЭТАПЕ 1 сетевое соединение между оболочками не реализуется.
