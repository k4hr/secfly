# ADR-0003: Проверка и версионирование сообщений

- Статус: принято
- Дата: 2026-07-21

## Решение

Все входные сообщения проверяются Zod-схемой оболочки и отдельной схемой содержимого. Поддерживаемая основная версия задаётся явно; неизвестная основная версия всегда отклоняется. Дополнительная версия принимается только политикой совместимости. Проверка не исправляет сообщения молча и возвращает безопасную русскую ошибку.

Технические идентификаторы остаются латинскими для совместимости, а пользовательские пояснения централизованы на русском языке.
