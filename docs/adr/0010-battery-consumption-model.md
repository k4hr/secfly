# ADR-0010: Модель расхода виртуального заряда

- Статус: принято
- Дата: 2026-07-21

## Решение

За каждый шаг заряд уменьшается пропорционально виртуальному времени: всегда применяется расход ожидания, а при движении добавляется отдельный расход. Значение ограничено диапазоном 0–100 процентов. При достижении нуля запуск переходит в `FAILED` с русским объяснением.

Модель детерминирована и намеренно не обладает физической точностью. Она не описывает аккумулятор, не прогнозирует остаточное время и не запускает возврат или посадку.
