# ADR-0007: Упрощённая синтетическая модель аппарата

- Статус: принято
- Дата: 2026-07-21
- Этап: ЭТАП 2

## Решение

Детерминированная предметная модель размещается в `apps/simulator`; `ground-control` только отправляет проверяемые команды и отображает полученное состояние. Модель описывает условное положение, высоту, скорость, направление и заряд виртуального гражданского аппарата.

Это прозрачная математическая демонстрация, а не аэродинамическая или сертификационная модель. Она не учитывает тягу, массу, инерцию, ветер, двигатели или полётный контроллер и не подключается к оборудованию. Все значения синтетические и не являются эксплуатационными характеристиками.

## Последствия

Одинаковые входы воспроизводимы; пользовательский интерфейс не владеет предметным состоянием. Результаты нельзя применять для настоящего полёта.

На ЭТАПЕ 2 интерфейс обращается прямо к проверяемому API Simulator через серверный прокси. `telemetry-gateway` не включается в путь только ради формального слоя: он понадобится на отдельном этапе моделирования связи.
