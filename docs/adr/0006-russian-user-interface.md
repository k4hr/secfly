# ADR-0006: Русский пользовательский интерфейс

- Статус: принято
- Дата: 2026-07-21

## Решение

Все смысловые пользовательские тексты SecFly пишутся на русском языке. Допустимые исключения: название SecFly, номер версии, технические адреса и обязательные программные идентификаторы. Отображаемые названия режимов, ошибок, уровней, состояний компонентов и действий хранятся в централизованном словаре `@secfly/ui`.

Испытания проверяют полноту словаря и ключевые тексты интерфейса. Технический код ошибки никогда не заменяет русское объяснение.
