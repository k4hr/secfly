# ADR-0004: Детерминированное виртуальное время

- Статус: принято
- Дата: 2026-07-21

## Решение

Предметная логика получает время через интерфейс `Clock`. `VirtualClock` начинает работу с явно переданного момента, движется только вперёд и не зависит от скорости компьютера. `SystemClock` использует системное время исключительно на инфраструктурной границе.

Прямые `Date.now()`, `performance.now()` и `new Date()` без аргумента автоматически запрещены в `safety-engine`, `state-estimator`, `protocol` и `event-log`.
