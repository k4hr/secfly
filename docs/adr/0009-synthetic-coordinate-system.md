# ADR-0009: Синтетическая система координат

- Статус: принято
- Дата: 2026-07-21

## Решение

Используется плоская локальная система: X и Y задают условное положение, Z представлена полем `altitude`. Условные единицы отображаются как метры только для удобства лабораторной демонстрации. Все числа конечны, высота неотрицательна.

Широта, долгота, адреса, географические карты и привязка к реальной местности отсутствуют. SVG показывает только собственную координатную сетку.
