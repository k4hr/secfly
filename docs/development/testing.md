# Испытания

`pnpm test` запускает модульные испытания, `pnpm test:coverage` формирует покрытие, `pnpm architecture:check` проверяет граф зависимостей и запрет системного времени, `pnpm check` выполняет полный обязательный набор. Сквозная конфигурация Playwright добавлена для проверки русской главной страницы.
