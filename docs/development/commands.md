# Команды разработки

- `pnpm dev` — запустить оболочки в режиме разработки.
- `pnpm build` — собрать все приложения и пакеты.
- `pnpm lint` — выполнить статический анализ.
- `pnpm format:check` — проверить форматирование.
- `pnpm format` — применить форматирование.
- `pnpm typecheck` — выполнить строгую проверку типов.
- `pnpm test` — запустить модульные испытания.
- `pnpm test:coverage` — сформировать отчёт о покрытии.
- `pnpm architecture:check` — проверить архитектурные ограничения.
- `pnpm check` — выполнить полный набор проверок.
- `pnpm clean` — удалить только формируемые артефакты рабочих областей.
